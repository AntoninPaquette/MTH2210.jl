MTH2210.jl - Fonctions pour le cours MTH2210

Le package MTH2210.jl contient toutes les fonctions Julia présentées dans le cadre du cours MTH2210 de l'École Polytechnique de Montréal.

Un gabarit Jupyter Notebook est disponible sur la page GitHub MTH2210.jl. Ce gabarit importe les librairies nécessaires pour le cours. Il est donc fortement recommandé d'employer ce gabarit pour la réalisation des laboratoires et des devoirs.

Cette documentation contient de l'information et des exemples minimaux pour:

  • Problèmes d'interpolation
  • Problèmes non-linéaires
  • Problème d'équations différentielles ordinaires

Une description de toutes les fonctions du cours est disponible à l'Index des fonctions.

Auteurs : Antonin Paquette et Alexis Montoison
